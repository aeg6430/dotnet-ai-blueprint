# Shadow examples (`*.cs.txt`)

These files are **non-compiling** examples intended for:

- Copilot/codegen anchoring (copy patterns)
- Human reference during reviews
- Avoiding "two truths" between docs and actual code

Recommended usage:

1. Copy the smallest matching example into your project.
2. Rename namespaces/types to your project conventions.
3. Keep the example style consistent with your analyzers and architecture tests.
